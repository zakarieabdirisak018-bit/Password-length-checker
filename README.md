# Password Length Checker

A lightweight and responsive web tool that analyzes password length in real-time and provides instant visual feedback on its strength.

---

### How does it work?

1. **Type in password:** Start typing your password into the input field.
2. **Weak (< 5 characters):** The background turns **red** with a text indicator showing **"Weak"**.
3. **Medium (5 to 7 characters):** The background turns **orange/yellow** with a text indicator showing **"Medium"**.
4. **Strong (8+ characters):** The background turns **green** with a text indicator showing **"Strong"**.

## Features

* ⚡ **Real-time validation:** Instant color change as you type.
* 🎨 **Visual feedback:** Clear color-coded system (Red, Orange, Green).
* 📱 **Responsive design:** Works flawlessly on desktop and mobile screens.
